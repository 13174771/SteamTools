# [Xamarin.Android](https://docs.microsoft.com/zh-cn/xamarin/android)
Xamarin.Android 向 .NET 开发人员公开了完整的 Android SDK。 使用 C# 或 F# 在 Visual Studio 中生成完全本机的 Android 应用。
